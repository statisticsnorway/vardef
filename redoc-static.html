variable-definitions

Variable definitions

List public variable definitions

Paginated list of all public variable definitions. Available without authentication. The list may be filtered using the available query parameters.

query Parameters
classification_uri
string <uri>

Return only variable definitions which reference this classification.

unit_type
string

Return only variable definitions which specify this Unit type.

subject_field
string

Return only variable definitions which are used in this subject field.

direct_person_identifying
boolean

Filter variable definitions based on whether they contain directly personally identifying information.

measurement_type
string
owner
string

Filter variable definitions by the division in Statistics Norway which owns them.

contact_person
string

Filter variable definitions by the contact person for them.

header Parameters
Accept-Language
string
Default: nb
Enum: "nb" "nn" "en"

Specify the language in which data will be presented.
